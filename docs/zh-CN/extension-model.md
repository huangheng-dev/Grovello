# 扩展模型

Grovello 通过公开、版本化的契约保持能力强大，同时避免供应商锁定。

- **连接器：**声明能力、风险、配置 Schema、幂等、健康、Webhook 与执行结果；密钥只以引用存在，不能进入模型上下文。
- **Agent：**声明版本、输入输出、允许工具、模型路由、评估集、成本上限与人工中断规则。
- **工作流：**确定性的 Temporal 定义，包含版本化输入/结果、重试、超时、审批、取消和补偿。
- **模板：**基于公开 Agent/工作流/连接器契约安装，不允许隐藏作者私有云依赖。
- **模型供应商：**对文本、结构化输出、Embedding、图片、音频、视频能力进行规范化适配。
- **MCP Server：**作为可选 Agent 工具入口；高风险工具仍必须通过 Grovello 授权和工作流策略。

第三方扩展必须声明外连域名、权限、数据用途、密钥需求、副作用、可逆性、限流、许可证与能力状态。安装扩展不等于自动授予执行权限。
