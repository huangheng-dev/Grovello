# 能力状态

| 状态 | 含义 |
| --- | --- |
| Foundation | 已有页面、产品定义、共享对象、架构边界与初始契约 |
| Simulated | 使用明确标注的种子或 Mock 数据，不产生外部能力声称 |
| Connected | 某供应商账号已通过契约、认证、健康、策略与沙箱测试 |
| Operational | 已验证端到端执行、失败处理、审计、衡量和运维手册 |
| Experimental | 通过显式开关使用，已声明限制与评估标准 |
| Planned | 已接受的架构项，但没有生产实现声明 |

当前仓库已经实现双语管理端、完整领域导航、初始 API 契约、核心数据库迁移、Temporal 工作流基础、LangGraph 决策图和连接器安全契约。外部渠道、身份、授权、分析投影和生产供应商账号尚未连接；界面示例数据均明确标记。
