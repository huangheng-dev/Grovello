# Code of Conduct

Grovello is committed to a respectful, inclusive, harassment-free community. Be constructive, evidence-based, and considerate. Respect consent, privacy, platform policy, security boundaries, and the people affected by growth automation.

Harassment, discrimination, threats, doxxing, deliberate disruption, or encouraging abusive outreach, identity deception, data theft, or policy evasion is not acceptable.

Before the first public release, the project owner must configure a private conduct reporting channel. Maintainers may remove content or participation that violates this code.
